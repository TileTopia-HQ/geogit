Quick Start

# Install (from source)
cargo install --path crates/geogit-cli

# Initialize a new repository
geogit init my-project
cd my-project

# Import a GeoPackage
geogit import GPKG:~/data/parcels.gpkg

# Create initial commit
geogit commit -m "Import parcels dataset"

# Create a branch for edits
geogit branch feature/update-zoning
geogit switch feature/update-zoning

# Edit data in QGIS (open the .gpkg file)
# Then check what changed:
geogit diff
geogit status

# Commit and merge
geogit commit -m "Update zoning classification"
geogit switch master
geogit merge feature/update-zoning

# Push to remote
geogit remote add origin git@github.com:org/geodata.git
geogit push origin master

Architecture

geogit-encoding

  • ColumnValue enum (Null, Bool, Integer, Float, Text, Blob)
  • Legend: schema evolution via column ID mappings
  • StoredFeature: MessagePack encode/decode
  • PathStructure: integer & msgpack-hash schemes
  • GpkgGeometry: GeoPackage Binary format

geogit-core

  • DatasetMeta: title, description, schema, path structure
  • FeatureDelta: Insert/Delete/Update with old/new values
  • DiffResult with summary statistics
  • Three-way merge with conflict detection

geogit-git

  • Repository: init, open, commit, push, pull, clone
  • TreeBuilder: write .table-dataset structures
  • Branch operations (create/delete/switch)
  • Merge with conflict detection & resolution
  • Diff (tree-to-tree and working directory)

geogit-wc

  • WorkingCopy trait (checkout, status, reset)
  • GeoPackageWorkingCopy implementation
  • ChangeTracker: SQL triggers on INSERT/UPDATE/DELETE
  • Automatic tracking table management

geogit-cli

  • 24 commands covering full Git+Kart workflow
  • Clap-derived argument parsing
  • File datasets & metadata management
  • Progress bars (indicatif)
  • Colored diff output

Storage Format

  • Standard Git repository (any hosting)
  • .table-dataset/ — row-level geospatial data
  • .file-dataset/ — versioned documents
  • meta/metadata.xml — ISO metadata
  • meta/license — license information
